# Scientific Analysis & Culinary Use

## Description
El almidón de yuca (o yuca agria) es el polvo extraído de la raíz. Es fundamental para la panadería colombiana (pandebono, pandeyuca, buñuelo), aportando elasticidad y textura "chiclosa" sin gluten.

## Culinary Uses
- **Pandebono:** Ingrediente base.
- **Buñuelo:** Mezclado con fécula de maíz.
