# Scientific Analysis & Culinary Use

## Description
El aceite vegetal (soya, girasol, palma) es el medio de cocción estándar para sofritos y frituras en Colombia.

## Culinary Uses
- **Sofritos:** Base de sabor.
- **Frituras:** Para patacones y tajadas.
