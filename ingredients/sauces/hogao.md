# Scientific Analysis & Culinary Use

## Description
El Hogao no es un ingrediente crudo, sino una "preparación madre" de la cocina colombiana. Es una salsa de tomate, cebolla junca (larga) y condimentos sofritos en aceite y color.

## Culinary Uses
- **Base:** Para sudados, fríjoles, arroces.
- **Salsa:** Encima de patacones o arepas.
