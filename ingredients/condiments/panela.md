# Scientific Analysis & Culinary Use

## Description
La panela es jugo de caña de azúcar deshidratado, sin refinar (sin centrifugar), conservando trazas de minerales que el azúcar blanco pierde. Es la bebida nacional (Aguapanela).

## Culinary Uses
- **Aguapanela:** Bebida caliente (con queso) o fría (con limón).
- **Endulzante:** Para café o postres.

## Health Benefits
### Minerales Traza
Aunque es 98% azúcar, contiene cantidades pequeñas pero medibles de hierro y calcio, a diferencia del azúcar refinado que es calorías vacías puras.
