# Scientific Analysis & Culinary Use

## Description
La leche es base de bebidas calientes (chocolate, café), postres (arroz con leche) y algunas sopas (changua).

## Culinary Uses
- **Chocolate Santafereño:** Preparado con leche.
- **Changua:** Caldo de leche con huevo.
