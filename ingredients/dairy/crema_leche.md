# Scientific Analysis & Culinary Use

## Description
La crema de leche es la grasa láctea concentrada. Es el toque final obligatorio del Ajiaco Santafereño y base de postres.

## Culinary Uses
- **Ajiaco:** Se añade en la mesa.
- **Postres:** Fresas con crema.
