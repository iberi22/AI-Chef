# Scientific Analysis & Culinary Use

## Description
La mantequilla es la grasa láctea batida. Se usa en panadería y para finalizar salsas.

## Culinary Uses
- **Arepas:** Untada encima.
- **Panadería:** Pandebonos, tortas.
